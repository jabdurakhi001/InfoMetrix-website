# InfoMetrix LLC Website

A professional, modern website for InfoMetrix LLC — a data-driven consulting firm helping operators turn data into actionable decisions and results.

![InfoMetrix Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## Features

- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices
- **Light/Dark Mode** — Toggle between themes with persistent preference storage
- **Modern Aesthetics** — Clean typography using DM Sans and Instrument Serif fonts
- **Smooth Interactions** — Animated FAQ accordion, hover effects, and scroll behaviors
- **Single Page Layout** — All content accessible via smooth-scrolling navigation

## Sections

1. **Hero** — Bold headline with CTAs and integration logos
2. **What We Unlock** — Four benefit cards (Financial Clarity, Process Design, AI & Automation, Logistics Operations)
3. **What We Do** — Six service categories with detailed offerings
4. **Who We Serve** — Industry focus cards and target audience description
5. **Selected Work** — Three case study previews
6. **How We Work** — Three-step consulting process visualization
7. **FAQ** — Interactive accordion with common questions
8. **Contact** — Contact form and company information
9. **Footer** — Navigation links and brand tagline

## Quick Start

### Option 1: Open Directly
Simply open `index.html` in your web browser.

### Option 2: Local Server
```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000`

## Deployment

### GitHub Pages
1. Go to repository **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Choose **main** branch and **/ (root)** folder
4. Click **Save**
5. Your site will be live at `https://jabdurakhi001.github.io/InfoMetrix-website/`

### Other Platforms
The site is static HTML/CSS/JS and can be deployed to:
- Netlify
- Vercel
- Cloudflare Pages
- Any static hosting service

## Customization

### Colors
Edit the CSS variables in the `:root` selector within `index.html`:

```css
:root {
    --accent-blue: #2563EB;
    --accent-green: #059669;
    --accent-orange: #EA580C;
    --accent-purple: #7C3AED;
    /* ... more variables */
}
```

### Content
All content is contained within `index.html`. Update text, links, and contact information as needed.

### Fonts
The site uses Google Fonts:
- **Instrument Serif** — Display headings
- **DM Sans** — Body text
